# BillEase Widgets

BillEase Widgets is a collection of reusable Flutter UI components designed to simplify the development of mobile applications such as POS systems, billing apps, and admin dashboards.

These widgets are lightweight, customizable, and built with Flutter's Material Design.

---

## Features

* Reusable Flutter UI components
* Designed for mobile billing and POS applications
* Easy to integrate into any Flutter project
* Clean and simple UI structure
* Modular widget-based architecture

---

## Widgets Included

The library currently includes the following widgets:

* **BE Button** – Custom reusable button widget
* **BE Calendar** – Date selection calendar component
* **BE DOB Input** – Date of birth input field
* **BE Header** – Custom page header widget
* **BE Icon Button** – Button with icon support
* **BE Icon Text Button** – Button with icon and text
* **BE Input Text** – Reusable text input field
* **BE Loading** – Loading indicator widget
* **BE Profile Pic** – Profile picture display widget
* **BE Snackbar** – Custom snackbar notification
* **BE Text Button** – Lightweight text button
* **BE TextView** – Styled text display widget

---

## Usage

Import widgets using the main export file.

Example:

import 'widgets/widgets.dart';

Then use any widget inside your Flutter UI.

Example:

BeButton(
text: "Submit",
onTap: () {},
)
